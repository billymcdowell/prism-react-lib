# Input Group

<SectionHeader
  title="Component Overview"
  description="Group input controls with add-ons, buttons, or text."
/>

Allows prepending or appending elements to an Input to provide context or actions.

## Demo

<Canvas />

## Anatomy
![Anatomy Diagram Placeholder: Annotated diagram showing: 1. Group, 2. Addon (Text/Icon), 3. Input]

1.  **InputGroup**: The container.
2.  **InputGroupAddon**: Container for side elements.
3.  **InputGroupInput**: The input field itself.
4.  **InputGroupText**: Static text (e.g., "https://").
5.  **InputGroupButton**: Action button (e.g., "Copy").

## Variants
-   **Start**: Addon at the start.
-   **End**: Addon at the end.
-   **Both**: Addons on both sides.
