## Behavior

### Interaction patterns
-   **Drag**: Mouse drag to change value.
-   **Click**: Jump to value on track.
-   **Keyboard**: Arrow keys adjust value by step.

<SectionHeader
  title="Usage Guidelines"

<Guidelines>
  <Dos>
    - **Range**: Use for selecting a value within a fixed range (volume, brightness).
    - **Approximation**: Use when exact values are less important than relative values.
  </Dos>
</Guidelines>
