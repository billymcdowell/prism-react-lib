## Anatomy
![Anatomy Diagram Placeholder: Annotated diagram showing: 1. Root, 2. List, 3. Item, 4. Trigger, 5. Content, 6. Link, 7. Viewport]

1.  **NavigationMenu (Root)**: Wrapper.
2.  **NavigationMenuList**: The list of items.
3.  **NavigationMenuItem**: Single list item.
4.  **NavigationMenuTrigger**: Button opening a dropdown.
5.  **NavigationMenuContent**: The mega-menu content.
6.  **NavigationMenuLink**: A standard link (styled).
7.  **NavigationMenuViewport**: The container that animates width/height for dropdowns.

## Behavior

### Interaction patterns
-   **Hover**: Opens on hover (desktop).
-   **Click**: Opens on click (touch).
-   **Delay**: Small delay to prevent accidental triggers.

### Screen reader support

### Technical considerations

-   Requires a `Viewport` component for the sliding animation effect.
