## Behavior

### Interaction patterns
-   **Click**: Selects the option.
-   **Arrow Keys**: Moves selection between options (standard radio behavior).

<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Radio Group component."
/>

<Guidelines>
  <Dos>
    - **Single Selection**: Use when user must pick exactly one option.
    - **Few Options**: Use for 2-5 options.
  </Dos>
  <Donts>
    - **Many Options**: Use for >5 options (use Select).
    - **Optional**: Use if selection is optional (use Checkbox or add "None" option).
  </Donts>
</Guidelines>
