## Anatomy
1.  **ScrollArea (Root)**: The wrapper.
2.  **Viewport**: The scrollable window.
3.  **ScrollBar**: The custom track/thumb.
4.  **Corner**: Handles the intersection of two scrollbars.

## Behavior

### Interaction patterns
-   **Scroll**: Works with wheel, touch, and drag.
-   **Hover**: Scrollbars can appear on hover (optional).

<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Scroll Area component."
/>

<Guidelines>
  <Dos>
    - **Navigation**: Use for sidebar navigation lists.
    - **Code Blocks**: Use for long code snippets.
  </Dos>
  <Donts>
    - **Global Scroll**: Replace the browser's main scrollbar (unless it's a full-screen app layout).
  </Donts>
</Guidelines>
