7.  **SelectLabel**: Group header.

## Behavior

### Interaction patterns
-   **Click**: Opens the list.
-   **Type**: Typing filters/highlights items.
-   **Arrow Keys**: Navigate items.

<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Select component."
/>

<Guidelines>
  <Dos>
    - **Many Options**: Use when there are more than 5 options (space saving).
    - **Forms**: Standard input for forms.
  </Dos>
  <Donts>
    - **Few Options**: Use for < 3 options (use Radio Group or Toggle Group).
  </Donts>
</Guidelines>

## Accessibility

### Screen reader support
-   **Role**: `combobox` or `listbox`.
-   **Value**: Announces selected value.
