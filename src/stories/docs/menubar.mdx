## Anatomy
![Anatomy Diagram Placeholder: Annotated diagram showing: 1. Menubar (Root), 2. Menu, 3. Trigger, 4. Content, 5. Items]

1.  **Menubar (Root)**: The horizontal bar.
2.  **MenubarMenu**: Wrapper for a single menu dropdown.
3.  **MenubarTrigger**: The button on the bar (e.g., "File").
4.  **MenubarContent**: The dropdown list.
5.  **MenubarItem**: Action items.
6.  **MenubarSub**: Submenus.

## Behavior

### Interaction patterns
-   **Click**: Opens the menu.
-   **Hover**: If one menu is open, hovering others opens them (roving).
-   **Keyboard**: Arrow keys navigate left/right between menus and up/down within menus.
