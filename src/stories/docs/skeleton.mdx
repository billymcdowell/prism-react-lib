<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Skeleton component."
/>

<Guidelines>
  <Dos>
    - **Match Layout**: Shape skeletons to match the content that will load (circle for avatar, rect for text).
    - **Progressive Loading**: Show layout immediately before data arrives.
  </Dos>
  <Donts>
    - **Spinners**: Don't mix skeletons and spinners in the same area (choose one).
  </Donts>
</Guidelines>
