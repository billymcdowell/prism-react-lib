## Anatomy
![Anatomy Diagram Placeholder: Annotated diagram showing: 1. Media, 2. Content (Title/Description), 3. Actions]

1.  **Item (Root)**: Container.
2.  **ItemMedia**: Icon or image.
3.  **ItemContent**: Wrapper for text.
4.  **ItemTitle**: Primary text.
5.  **ItemDescription**: Secondary text.
6.  **ItemActions**: End-aligned buttons.

## Variants
-   **Default**: Transparent background.
-   **Outline**: Bordered.
-   **Muted**: Slight background color.
