# Collapsible

<SectionHeader
  title="Component Overview"
  description="An interactive component which expands/collapses a panel."
/>

Used to hide complex or secondary content while keeping it accessible.

## Demo

<Canvas />

## Anatomy
![Anatomy Diagram Placeholder: Annotated diagram showing: 1. Root, 2. Trigger, 3. Content]

1.  **Collapsible (Root)**: State manager.
2.  **CollapsibleTrigger**: The button that toggles the state.
3.  **CollapsibleContent**: The panel that expands/collapses.

## States
-   **Open**: Content is visible.
-   **Closed**: Content is hidden (height: 0).

## Behavior

### Interaction patterns
-   **Click**: Toggles the open/closed state.

### Animation
-   Typically animates the `height` or `grid-template-rows` from 0 to auto.

<Guidelines>

## Accessibility

### Screen reader support
-   **Trigger**: `aria-expanded` indicates state. `aria-controls` points to content.
-   **Content**: `region` role (optional but good for large sections).
