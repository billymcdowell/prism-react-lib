# Popover

<SectionHeader
  title="Component Overview"
  description="Displays rich content in a portal, triggered by a button."

Used for small overlays that are not modal (don't block the rest of the UI).

## Behavior

### Interaction patterns
-   **Click**: Toggles open/closed.
-   **Click Outside**: Closes the popover.
-   **Focus**: Focus moves into the popover (unlike Tooltip).

<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Popover component."
/>

<Guidelines>
  <Dos>
    - **Filters**: Use for complex filter forms.
    - **Color Pickers**: Use for small interactive tools.
    - **Date Pickers**: Standard container for calendars.
  </Dos>
  <Donts>
    - **Critical Info**: Hide critical info in popovers.
    - **Large Content**: Use for large amounts of content (use Dialog/Sheet).
  </Donts>
</Guidelines>

## Accessibility

### Screen reader support
-   **Role**: `dialog`.
-   **Focus**: Traps focus while open (usually).
