## Anatomy

1.  **DropdownMenu (Root)**: Wrapper.
2.  **DropdownMenuTrigger**: Button that opens the menu.
3.  **DropdownMenuContent**: The popup container.
4.  **DropdownMenuItem**: Action items.
5.  **DropdownMenuCheckboxItem**: Toggleable items.
6.  **DropdownMenuRadioGroup**: Single select items.

## Behavior

### Interaction patterns
-   **Click**: Opens on click (unlike Tooltip which is hover).
-   **Selection**: Clicking an item executes action and closes menu.
