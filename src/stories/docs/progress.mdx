<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Progress component."
/>

<Guidelines>
  <Dos>
    - **Deterministic**: Use when the progress value is known (0-100%).
    - **Loading**: Use for file uploads or long processes.
  </Dos>
  <Donts>
    - Use for indeterminate loading (use Spinner/Skeleton).
  </Donts>
</Guidelines>

## Accessibility

### Screen reader support
-   **Role**: `progressbar`.
-   **Value**: `aria-valuenow`, `aria-valuemin`, `aria-valuemax`.
