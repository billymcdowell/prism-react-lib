# Field

<SectionHeader
  title="Component Overview"
  description="A flexible container for form fields, labels, and helper text."
/>

Provides a consistent structure for inputs, labels, descriptions, and error messages.

<Canvas />

## Anatomy
![Anatomy Diagram Placeholder: Annotated diagram showing: 1. Field (Wrapper), 2. Label, 3. Content (Input), 4. Description, 5. Error]

1.  **Field (Root)**: Flex container.
2.  **FieldLabel**: Accessible label.
3.  **FieldContent**: Wrapper for the input component.
4.  **FieldDescription**: Helper text.
5.  **FieldError**: Validation error message.

## Variants
-   **Vertical**: Stacked label, input, description (Default).
-   **Horizontal**: Label and input side-by-side (Switch, Checkbox).

<SectionHeader

<Guidelines>
  <Dos>

    - **Error Handling**: Use `FieldError` to display validation messages.
