## Components Used
This layout utilizes:
-   **Sidebar**: For navigation and folder structure.
-   **Breadcrumb**: For location tracking.
-   **Table**: For the list view of files.
-   **Grid**: Custom grid using Cards for the thumbnail view.
-   **DropdownMenu**: For file actions.
-   **Dialog/Sheet**: (Potential integrations for details).

## Features
-   **View Toggle**: Switch between Grid and List views.
-   **Navigation**: Sidebar and Breadcrumbs.
-   **Search**: Header search bar.
