# Pagination

<SectionHeader
  title="Component Overview"
  description="Pagination with page numbers, previous/next, and truncation."
/>

Allows users to navigate through large sets of data split into pages.

6.  **PaginationEllipsis**: Truncation indicator.

## Behavior

### Interaction patterns
-   **Click**: Navigates to the selected page.
-   **Disabled**: Previous/Next are disabled at boundaries.

<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Pagination component."
/>

<Guidelines>
  <Dos>
    - **Large Lists**: Use when data exceeds 20-50 items.
    - **Current Page**: Clearly indicate the current page.
  </Dos>
  <Donts>
    - Use for infinite scroll (unless as a fallback).
  </Donts>
</Guidelines>

## Accessibility

### Screen reader support
-   **Role**: `navigation`.
-   **Label**: `aria-label="pagination"`.
-   **Current**: `aria-current="page"` on the active link.
