## Behavior

### Interaction patterns
-   **Focus Trap**: Focus is trapped within the dialog while open.
-   **Scroll Lock**: Background scrolling is disabled.
-   **Dismiss**: Esc key or clicking overlay closes the dialog.

<SectionHeader
  title="Usage Guidelines"
  description="Best practices for when and how to use the Dialog component."
/>

<Guidelines>
  <Dos>
    - **Focused Tasks**: Use for forms, confirmations, or critical information requiring attention.
    - **Simplicity**: Keep dialog content simple.
  </Dos>
  <Donts>
    - **Nested Dialogs**: Avoid nesting dialogs (use a step-by-step wizard or a new page).
    - **Full Pages**: Don't put an entire page's worth of content in a dialog (use a Sheet/Drawer).
  </Donts>
</Guidelines>
